## Instalación y Desarrollo

Este proyecto de portfolio ha sido desarrollado con Angular 21. StandAlone

1. Instalar dependencias:
   ```bash
   npm install
   ```

2. Iniciar servidor de desarrollo:
   ```bash
   npm start
   ```

3. Generar build de producción:
   ```bash
   npm run build
   ```

4. Arrancar Proyecto:
   ```bash
   ng serve
   ```
